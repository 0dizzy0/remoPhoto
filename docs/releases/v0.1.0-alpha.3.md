## 主要功能

### 本地相册

- 通过 Android Storage Access Framework 添加多个图片目录，并持久化目录访问授权。
- 扫描 PNG、JPG/JPEG、GIF、WebP 和 BMP 图片。
- 按子目录生成层级相册，支持重新扫描和删除仓库。
- 大仓库扫描使用 WorkManager、磁盘 spool、批量写入和事务切换，降低长时间扫描的内存压力。

### 相册与图片浏览

- 相册网格/列表展示、分页和排序。
- 按名称模糊定位相册。
- 图片网格/列表浏览和排序。
- 全屏浏览支持滑动、点击区域、缩放、音量键、鼠标滚轮和自动播放。
- 支持创建、编辑和删除分类，并关联本地或已连接的远程相册。

### 局域网远程相册

- 通过 HTTP 在局域网共享本设备的本地相册。
- 通过 mDNS 自动发现设备，也可以手动输入 IP 和端口。
- 支持远程相册元数据同步、分页、封面、图片浏览和本地缓存。
- 支持远程设备离线提示以及网络恢复后的重新连接。
- 不会把其他设备的远程相册再次共享出去。

### 设置与数据维护

- 支持主题、深色背景、高对比度、排序、每页数量和播放间隔等设置。
- 支持数据库与设置的导入导出。
- 导出内容可以包含远程连接元信息，但不包含远程凭据和图片缓存。
- 支持统计并清理远程缩略图和原图缓存。
- 扫描、同步、远程服务和备份流程保留可观测日志。

## 安装与升级

1. 仅从项目正式 Release 页面获取 APK，并核对发布页提供的 SHA-256。
2. 本版本要求 Android 10 或更高版本。
3. 安装 APK 时，Android 可能要求为当前文件来源临时开启“允许安装未知应用”。
4. 从内部测试包升级前，建议先使用应用内导出功能备份数据。
5. 只有使用同一签名密钥构建的旧版本才能覆盖安装；签名不同的测试包需要先卸载，卸载会清除应用数据。

## 已知限制与安全提示

- 本版本是 Alpha，不建议作为唯一的图片索引或数据备份方案。
- 远程相册使用明文 HTTP，当前没有 TLS 或访问令牌。只能在本人可控的可信局域网使用，不要在公共 Wi-Fi 或公网暴露服务。
- 数据库导出文件未加密，可能包含目录、相册、分类名称和局域网连接地址等敏感元数据。请妥善保管，不要直接公开上传。
- 暂不支持 SMB 电脑共享仓库、视频播放、多用户权限管理和公网访问。
- 大仓库扫描可能受到不同厂商后台限制影响；进程被系统终止后，扫描任务可能重新遍历目录。
- 当前自动化测试覆盖较少，主要发布回归仍依赖真机手工验证。
- targetSdk 当前为 35，尚未完成 Android 16/API 36 的专项兼容验证。
