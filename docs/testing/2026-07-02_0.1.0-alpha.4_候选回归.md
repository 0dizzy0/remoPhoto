## 3. 新增自动化覆盖

### 相册规则

- 分页覆盖负数、0、1、`P`、`P+1` 和跨多页边界。
- 名称定位覆盖 trim、忽略大小写、Unicode 和无匹配。
- 六种相册排序均验证主排序键和确定性名称/ID 次级顺序。
- 相册树子节点递归排序。

### 备份输入边界

- manifest 当前版本、非法版本、未来版本和数据库版本不一致。
- 无 manifest 的旧备份保持兼容。
- ZIP 条目白名单和 `settings/` DataStore 路径。
- 未知条目、路径穿越、重复条目、空 ZIP 和损坏 ZIP 拒绝。
- 常见 ZIP local/empty/spanned signature 识别。

### Room migration

真机测试创建 v4 数据库后按历史结构重建 v3 `albums` 表，写入两张相册和两张图片，再由 Room 执行 `MIGRATION_3_4`：

- 有图片相册的 `last_modified` 回填为最大图片修改时间 `20`。
- 空相册回填为 `0`。
- 数据库版本升级到 4。
- Room 最终 schema 校验通过。
