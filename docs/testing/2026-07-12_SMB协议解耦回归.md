# 2026-07-12 SMB 协议解耦（M1）回归记录

## 结论

**本地门禁通过，HTTP 双机 Smoke 待设备接入。** M1 已让业务同步 UseCase 脱离 HTTP DTO/API，并保持现有 HTTP Room 标识和图片 URL 不变。

## 改动摘要

- 新增协议无关 `RemoteAlbumRecord`、`RemoteMediaRecord`、`RemoteMediaRef`、错误类别和 `RemoteCatalogSource`。
- 新增 `HttpRemoteCatalogSource`，HTTP DTO 只存在于 `RemoteHttpClient` 与 adapter 内。
- 新增 `RemoteSourceRouter`；SMB 未注册时明确返回 `UNSUPPORTED_PROTOCOL`，不会错误回落到 HTTP。
- `SyncRemoteRepositoryUseCase` 使用 opaque key，并继续生成与改动前完全相同的 HTTP 相册路径、封面 URL 和原图 URL。
- 远程操作取消立即上抛，不进入重试。
- 清理远程链路 host、port、URL、路径和仓库名日志；异常只记录稳定类别。
