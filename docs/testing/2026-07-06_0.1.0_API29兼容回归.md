## 4. REL-012 复现

Android 10 DownloadsProvider 对 tree URI 的直接查询记录 `Unsupported Uri`，应用随后把 URL 解码后的完整 document ID 用作名称：

- 仓库名：`raw:/storage/emulated/0/Download/RemoPhotoApi29`
- 相册名：`raw:/storage/emulated/0/Download/RemoPhotoApi29/AlbumA`

扫描和浏览功能未失败，但该显示不适合作为正式版最低系统体验。

## 5. 修复与复测

修复内容：

1. tree URI 查询显示名之前，先转换为 `buildDocumentUriUsingTree(...)` 对应的 document URI。
2. provider 查询失败时，统一从 `raw:`、`primary:` 等 document ID 提取末级目录名。
3. 回退日志只记录 authority 与是否为 tree URI，不记录名称、URI 或路径。
4. 增加 raw tree、raw child、primary/Unicode、加号保留和空输入 5 条 JVM 测试。
