## 3. 执行步骤

```powershell
adb -s 400F5QF1KF00000 push .cache\release-verify-v0.1.0-beta.1\remoPhoto-0.1.0-beta.1.apk /data/local/tmp/remophoto-beta1.apk
adb -s 400F5QF1KF00000 shell pm install -r /data/local/tmp/remophoto-beta1.apk
adb -s 400F5QF1KF00000 push app\build\outputs\apk\release\app-release.apk /data/local/tmp/remophoto-010.apk
adb -s 400F5QF1KF00000 shell pm install -r /data/local/tmp/remophoto-010.apk
adb -s 400F5QF1KF00000 shell am start -W -n com.remophoto/.MainActivity
```

安装前对 Beta APK 与候选 APK 均执行 `apksigner verify --verbose --print-certs`，两者 v2 签名通过且签名证书一致。
