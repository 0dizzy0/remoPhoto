## 3. 执行命令

Gradle UTP 在本机超时但设备端未启动测试进程，因此改用 ADB 直接安装和运行 instrumentation：

```powershell
adb -s 1e6cd7cc push app\build\outputs\apk\debug\app-debug.apk /data/local/tmp/remophoto-debug.apk
adb -s 1e6cd7cc shell pm install -r -t /data/local/tmp/remophoto-debug.apk
adb -s 1e6cd7cc push app\build\outputs\apk\androidTest\debug\app-debug-androidTest.apk /data/local/tmp/remophoto-debug-androidTest.apk
adb -s 1e6cd7cc shell pm install -r -t /data/local/tmp/remophoto-debug-androidTest.apk
adb -s 1e6cd7cc shell am instrument -w -r com.remophoto.debug.test/androidx.test.runner.AndroidJUnitRunner
```

- `DatabaseExporter.importDatabase` 增加测试用 `restartProcess=false` 参数，默认值为 `true`，生产导入成功后仍会重启进程。
