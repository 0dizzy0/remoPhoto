# 0.1.0 候选双机远程 Smoke 记录

## 1. 结论

`0.1.0` 候选在两台真机完成双机远程 Smoke：共享端开启 HTTP 前台服务，客户端通过 mDNS 自动发现、连接远程仓库、同步远程相册、进入远程图片网格并全屏打开远程图片。服务停止后客户端 HTTP 访问失败，服务恢复后 `/api` 再次可用。Crash buffer 为空，远程流程 Release 日志隐私扫描通过。

本次使用本地 SAF Smoke 创建的公开测试图片，不使用用户私人图片。截图、UI dump、logcat、Crash buffer 和应用文件日志保存在 Git 忽略目录。
