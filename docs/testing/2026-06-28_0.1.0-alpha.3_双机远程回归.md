## 4. 关键验证结果

### 发现、同步和分页

- mDNS 在 6 秒内发现 1 台远程设备。
- 连接后创建单一远程仓库。
- 首次进入同步到 5244 个相册，共 263 页。
- 从首页跳到末页成功；末页可见 4 个相册，和首页无重复。

### 图片和缓存

- 远程相册图片数量、网格和全屏原图显示正常。
- 二次进入相册可立即展示持久化元数据和缩略图。
- 初始候选包暴露 REL-009：全屏 `ZoomableImage` 使用全局 Coil `image_cache`，设置页原图缓存始终为 0 B。
- 修复后全屏远程 URL 显式使用 `remoteImageLoader/remote_image_cache`；最终候选包打开一张远程全屏原图后，设置页原图缓存从 0 B 增至 3.1 MB。
- 执行“清除远程缓存”后，缩略图和原图统计均回到 0 B；Coil `journal*` 索引元数据不再计入用户可清理缓存。

### 离线和恢复

- 关闭客户端 Wi-Fi 后，远程缓存仍可浏览。
- 对远程仓库执行一次失败访问后，卡片从已连接更新为断开/错误。
- 离线时分类保留，但不显示远程相册。
- 恢复 Wi-Fi 后，远程服务健康检查返回 200。
- 重新进入缓存页不会自动把卡片改回已连接；点击页面的显式刷新后恢复连接。
- 恢复后仍只有 1 个远程仓库，分类关联重新显示。

- 实际 DEBUG 级日志：0。两处放大镜符号属于 INFO 级 mDNS 消息正文。
