- Crash buffer 为空。

## 5. 有效备份恢复

1. 在 Beta 候选中导出有效 Backup-v1 ZIP，文件大小 2627 bytes。
2. 删除分类哨兵，将每页相册数量改为 `10`。
3. 从系统文件选择器导入刚导出的 ZIP。
4. 应用按设计结束当前进程；重新冷启动耗时 `272 ms`。
5. 分类 `BetaUpgradeSentinel` 恢复，每页相册数量恢复为 `50`。
6. Crash buffer 为空。

导入失败回滚的独立注入证据见[Beta 准备：导入失败回滚](2026-07-03_Beta准备_导入失败回滚.md)。
