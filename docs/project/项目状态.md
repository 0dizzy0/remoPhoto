## 当前能力

### 本地仓库

- 添加、删除和重新扫描本地图片仓库。
- 使用 SAF 授权访问 Android 10+ 文件目录。
- 支持 PNG、JPG/JPEG、GIF、WebP、BMP。
- 子目录自动映射为层级相册。
- 大仓库扫描已引入 WorkManager、磁盘 spool、事务切换和阶段化进度。

### 相册与图片浏览

- 相册列表支持网格/列表展示、分页、相册排序和名称模糊定位。
- 相册内图片支持网格/列表展示和排序。
- 全屏浏览支持滑动、点击区域、音量键、鼠标滚轮、缩放和自动播放。
- 分类支持创建、编辑、删除和批量关联相册。

### 远程仓库

- 本设备可通过局域网 HTTP 服务共享本地相册。
- 支持 mDNS 自动发现和手动 IP 添加。
- 远程相册元数据可同步到本地 Room 缓存。
- 远程相册支持分页、封面、分类关联和离线状态处理。
- 远程仓库仅面向可信局域网，不承诺公网访问安全。

### 设置与维护

- 支持主题、深色背景类型、高对比度、排序、分页数量、播放间隔等设置。
- 支持数据库导入导出；远程连接元信息可导出，凭据不导出。
- 远程缩略图和原图缓存可统计和清理。
- 关键路径使用 `AppLogger` 记录可观测日志。

## 当前发布建议

当前公开版本为 `0.1.0-beta.1`，对应 `versionCode = 5` 和 Git tag `v0.1.0-beta.1`，已完成发布。当前开发候选为 `0.1.0`，对应 `versionCode = 6`，尚未公开发布。

Beta 发布定位：

- 面向作者和早期用户试用。
- 允许存在已知限制。
- 不承诺 Play Store 上架质量。
- 不承诺 SMB、视频、多用户、HTTPS 或公网访问。

下一目标是完成 `0.1.0` 正式发布门禁并公开发布。额外 Beta 或 RC 只在出现高风险修复时增加，具体门禁见[Beta 与正式版 Release 计划](../releases/09_Beta与正式版Release计划.md)。
