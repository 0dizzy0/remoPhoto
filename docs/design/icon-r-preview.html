RemoPhoto 图标初版

方向：抽象 R + 相册/远程仓库语义。优先保证启动器大图标有个性，小尺寸也能辨认。

建议先看 32px 和 16px 的识别度；如果小尺寸还能看出 R 的走势，这个方向就适合继续做 Android adaptive icon。

推荐 V1

R / 相册卡片

几何 R 由相册页角和路径组成，右下角的绿色卡片代表已整理的照片集合。

V2

R / 远程连接

保留抽象 R，用一条轻量弧线表达远程访问和同步感，整体更偏工具产品。

V3

R / 胶片仓库

R 的竖线融合成胶片边孔，照片仓库感更强，但比 V1 稍复杂。

文件内所有图形都是 SVG，可直接复制到 Android VectorDrawable 或作为 adaptive icon 的 foreground 继续拆分。当前更推荐 V1：功能识别最直接，形状也比普通字母 Logo 更有产品记忆点。
