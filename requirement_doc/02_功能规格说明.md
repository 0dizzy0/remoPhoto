# remoPhoto 功能规格

更新时间：2026-06-23

## 2. 仓库管理

### A1. 本地仓库

用户可以通过系统目录选择器添加本地图片目录。

验收要点：

- 使用 SAF 获取并持久化目录授权。
- 可添加多个仓库。
- 仓库卡片显示名称、路径摘要、图片数量、扫描状态。
- 可删除仓库及相关索引。
- 可手动重新扫描。
- 无仓库时仍能进入添加入口。

### A2. 扫描与索引

扫描负责递归读取授权目录并写入 Room 索引。

验收要点：

- 支持 PNG、JPG/JPEG、GIF、WebP、BMP。
- 子目录自动生成相册。
- 大仓库扫描不把所有图片对象常驻内存。
- 使用前台 WorkManager 承载长任务。
- 扫描成功后事务切换索引，失败或取消时保留旧索引。
- UI 展示阶段性进度：目录遍历、发现图片、写入索引等。
- 日志记录 repoId、阶段、耗时、数量、失败原因。

### A3. 远程仓库入口

用户可以添加局域网内其他设备共享的远程仓库。

验收要点：

- 支持 mDNS 自动发现。
- 支持手动输入 IP 和端口。
- 过滤本机服务，避免添加自身。
- 已存在设备重复添加时给出明确处理。
- 远程仓库状态包括已连接、断开、错误等。

## 3. 相册列表

### B1. 层级相册

每个子目录映射为相册，嵌套目录形成父子层级。

验收要点：

- 仓库列表和仓库内相册列表状态清晰。
- 点击有子相册的相册进入下一层。
- 系统返回键回到上一层或上一级页面。
- 切换仓库时不闪现旧仓库相册。

### B2. 分页

相册列表按设置的每页数量分页。

验收要点：

- 0、1、20、21、40、41、5225 等边界数量计算正确。
- 远程仓库总页数变化时分页栏响应式更新。
- 大型远程仓库可跳转到末页。

### B3. 排序

相册列表排序独立于相册内图片排序。

当前排序维度：

- 名称升序/降序。
- 图片数量升序/降序。
- 修改时间升序/降序。

验收要点：

- 修改相册列表排序不影响图片排序。
- 排序设置持久化。
- 同值时使用名称作为稳定次级排序。

### B4. 名称模糊定位

用户输入相册名称片段，应用定位到第一个命中相册所在页。

验收要点：

- 不过滤列表，只跳转页码。
- 大小写不敏感。
- 无命中时给出提示。
- 定位后列表总数不变。

### B5. 分类关联

用户可把相册加入分类。

验收要点：

- 支持长按多选并批量加入分类。
- 远程相册重同步后分类关联保持。
- 远程仓库离线时，分类中不显示该远程仓库的相册。

## 4. 图片浏览

### C1. 相册内图片列表

验收要点：

- 支持网格/列表模式。
- 图片缩略图懒加载。
- 支持 GIF/WebP 动图。
- 空相册展示明确空状态。
- 远程相册进入时按需同步图片元数据。

### C2. 全屏浏览

验收要点：

- 支持左右滑动翻页。
- 支持点击左/右区域翻页。
- 支持音量键翻页，可在设置关闭。
- 支持蓝牙/OTG 鼠标滚轮翻页。
- 支持双指缩放和双击 FIT/CROP 切换。
- 缩放状态下滚轮优先缩放，非缩放状态下滚轮翻页。
- 切图时避免往返闪烁和状态栏跳动。

### C3. 自动播放

验收要点：

- 可设置播放间隔。
- 进入全屏后可启动/暂停。
- 离开全屏或用户手动操作时状态可控。

## 5. 分类管理

验收要点：

- 分类可创建、重命名、修改颜色、删除。
- 删除分类会清理相册关联。
- 分类页显示本地相册和当前已连接远程仓库相册。
- 分类为空时展示空状态，不误导用户添加仓库。

## 6. 设置与维护

### E1. 常规设置

- 主题：浅色、深色、跟随系统。
- 深色背景：自动、OLED、LCD。
- 高对比度。
- 图片默认排序。
- 相册列表排序。
- 每页相册数量。
- 自动播放间隔。
- 音量键翻页开关。

### E2. 远程服务

- 开启/关闭远程访问。
- 配置端口和设备名称。
- 服务重启、自愈和后台保活。
- mDNS 注册和发现。

### E3. 缓存

- 显示本地索引统计。
- 显示远程缩略图缓存和原图缓存。
- 支持清除远程缓存。
- 统计失败时显示明确错误。

### E4. 导入导出

- 导出 Room 数据库快照、DataStore 设置和 manifest。
- 导入前执行白名单解压、完整性检查和版本检查。
- 导入失败时恢复旧数据库和设置。
- 远程连接凭据不导出。

## 7. 远期功能

以下功能不属于首个 alpha：

- SMB 仓库。
- 视频播放。
- 多用户权限。
- HTTPS/TLS 或访问令牌。
- 云端同步。
- Play Store 发布材料。
