## 九、风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| SAF 权限在部分 ROM 上兼容性问题 | 中 | 高 | Phase 0 提前适配主流厂商 ROM（MIUI、ColorOS 等） |
| 万级图片扫描性能不达标 | 中 | 高 | 分批扫描 + 索引优化 + 后台 Service |
| Coil GIF 动图内存/性能问题 | 中 | 中 | 限制同时播放 GIF 数量，全屏不可见时暂停 |
| mDNS 在部分路由器/ROM 不可用 | 高 | 中 | 提供手动输入 IP 的降级方案 |
| NanoHTTPd 后台被杀 | 中 | 中 | 前台 Service 保活，通知栏显示"远程服务运行中" |
| 大尺寸原图 OOM | 中 | 高 | Coil 降采样 + Bitmap 复用池 + 内存 LRU |
| 多用户数据安全（SQLite 明文） | 低 | 高 | 密码 SHA-256+Salt 哈希，敏感数据加密存储 |
